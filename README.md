# Silent Time

> 记录你远离干扰的纯净时间。

---

## 是什么

Silent Time 是一个纯静态网页工具，用来追踪你手机处于"无声状态"的总时长——也就是你远离抖音、B站、网易云音乐等有声 App 的时间。

它不做实时监控，而是利用浏览器的**音频焦点机制**来感知干扰：只要你打开了会发出声音的 App，网页音频就会被系统抢占，计时立刻停止。

---

## 核心原理

网页在后台循环播放一段白噪音（https://elckemvmphbjjlpzgoqy.supabase.co/storage/v1/object/public/white-sound/p1.mp3），占住系统音频焦点。

- **音频播放中** → 正常计时，累加纯净专注时长

- **音频被抢占**（打开抖音/B站等）→ 触发 `onpause` 事件，立即停止计时，停止播放，显示"检测到干扰"

- **打断后不自动恢复**，音频保持停止，不抢回焦点，等待用户主动操作

- **用户回到页面** → 手动点击"继续专注"，重新从头播放音频，开始新一段计时

锁屏、切后台不影响计时，只有有声 App 抢走焦点才会打断。

## 功能

| 功能 | 状态 |
|---|---|
| 音频焦点计时（锁屏不中断） | ✅ |
| 检测到干扰时停止计时 | ✅ |
| 今日累计专注时长（实时） | ✅ |
| 今日时间轴（专注时段可视化） | ✅ |
| 数据持久化（localStorage） | ✅ |

---

## 使用方式

1. 用手机浏览器打开页面（建议 Chrome / Safari）
2. 点击**「开始专注」**，允许音频播放
3. 把手机放下，正常做事
4. 打开抖音/B站等 App 时，页面会自动检测并停止计时
5. 回到页面后点击**「继续专注」**重新开始



---

## 技术栈

- **前端：** 原生 HTML / CSS / JavaScript（无框架）
- **样式：** Tailwind CSS（CDN）
- **音频：** url 获取白噪音，循环播放
- **存储：** localStorage（无后端、无数据库）
- **部署：**  GitHub Pages
